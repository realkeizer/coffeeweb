CoffeeWeb will consist of three major workflows when completed:

First, the components and fonts library. These will contain pre-packaged modules of code that are fully cross compatible,
so that all fonts and components are normalised to one another. This is important to the functionality of the program.

Second, the template generator. This will be a Python backend program that constructs full project structures and adds
components to relevant .html files based on "seeds".

Thirdly, a web interface. This will allow a user to either manually select components/fonts or to input a seed, and the
generator will run automatically to create the project.
